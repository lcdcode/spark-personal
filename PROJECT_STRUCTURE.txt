Features Implemented:
✅ Hierarchical Notes with Markdown support
✅ Spreadsheets with formula engine
✅ Code Snippets with syntax highlighting
✅ Backup Management with auto-backup
✅ Theme System (5 themes)
✅ Configuration System (YAML)
✅ Search Functionality
✅ Demo Data for onboarding

Database Schema:
- notes         (id, title, content, parent_id, timestamps)
- spreadsheets  (id, name, data, timestamps)
- snippets      (id, title, code, language, tags, timestamps)

Dependencies:
- PyQt6 >= 6.6.0
- PyYAML >= 6.0
- Pygments >= 2.17.0
- Markdown >= 3.5.0
